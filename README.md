# Commit-Classifier-in-Version-Control-System
Challenge:  build a model capable of classifying build account activities from their activity patterns as identified from metrics collected about code commits. Major points: how to deal missing data ,how to deal outliers ,how to deal correlation among features ,feature extraction, testing and validating
